# 用友ERP沙盘模拟详细说明文件
## 一、课程概述
### 1.1 开课背景
高等教育创新长期受到政府、社会、教育研究及管理者乃至全社会的关注。我国相关政策持续推动创新创业教育发展：
- 党的十七大提出“提高自主创新能力，建设创新型国家”和“促进以创业带动就业”的发展战略；
- 2012年教育部印发《普通本科学校创业教育教学基本要求(试行)》，将创新创业教育列为必修课，规范高校创新创业教育教学；
- 2015年《国务院办公厅关于深化高等学校创新创业教育改革的实施意见》颁布，明确改革的指导思想、基本原则和总体目标；
- 2016年《国家创新驱动发展战略纲要》发布，明确企业、科研院所、高校等各类创新主体功能定位。

在此背景下，高等院校将创新创业课程纳入学分管理和本科生人才培养方案，构建依次递进、有机链接、科学合理的创新创业课程群，《创新实践及科研训练》（创新创业实践）应运而生。

### 1.4 考核与成绩评定
总评成绩=平时表现20%+模拟企业运营70%+实践报告10%
1. **平时表现（20%）**：涵盖出勤情况、上课互动表现、作业完成质量等；
2. **模拟企业运营（70%）**：以小组为单位，根据模拟企业运营过程中的运行控制记录、小组排名等指标评定；
3. **实践报告（10%）**：包括调研过程形成的调研报告，以及个人参与课程后的学习报告。

## 二、沙盘模拟核心框架
本次沙盘模拟以用友ERP沙盘为基础，通过模拟企业全流程运营，提升参与者的综合管理能力。核心流程包括：组织准备工作→模拟企业概况了解→初始状态设定→熟悉企业运营规则→实际模拟训练。

## 三、组织准备工作
### 3.1 人员定位
参与者需分组扮演企业不同岗位角色，各司其职并协同配合，具体岗位职责如下：
| 岗位 | 核心职责 |
| ---- | ---- |
| 总经理 | 制定发展战略、分析竞争格局、确定经营指标、制定业务策略、全面预算管理、协调管理团队、分析企业绩效、考评管理业绩、管理授权与总结 |
| 财务主管 | 日常财务记账和登账、向税务部门报税、提供财务报表、日常现金管理、制定企业融资策略、控制成本费用、调度资金与管理风险、建立财务制度与风险管理、财务分析与协助决策 |
| 营销主管 | 市场调查分析、制定市场进入策略、规划品种发展策略、制定广告宣传策略、编制销售计划、争取定单与谈判、签订合同与过程控制、按时发货与应收款管理、销售绩效分析 |
| 生产主管 | 产品研发管理、管理体系认证、固定资产投资、编制生产计划、平衡生产能力、生产车间管理、产品质量保证、成品库存管理、产品外协管理 |
| 供应主管 | 编制采购计划、与供应商谈判、签订采购合同、监控采购过程、到货验收、仓储管理、采购支付决策、与财务部协调、与生产部协同 |

### 3.2 职能定位
企业运营需明确各中心职能分工，形成协同运作体系：
- 财务中心：财务总监、财务助理
- 生产中心：运营总监、生产主管
- 物流中心：采购总监、供应主管
- 营销与规划中心：营销总监
- 最高决策层：总裁CEO

### 3.3 核心要求
参与者需秉持以下理念开展模拟运营：
- 心态开放：积极接受新思维、新方法，勇于尝试；
- 亲历亲为：主动参与各项运营环节，积累实践经验；
- 团队协作：各岗位密切配合，共同实现企业目标；
- 换位思考：理解不同岗位的职责与难点，提升协同效率。

## 六、企业运营规则
### 6.1 市场划分与市场准入
#### 6.1.1 市场层级
分为本地市场、区域市场、国内市场、亚洲市场、国际市场五个层级，企业初始仅运营本地市场。

#### 6.1.2 市场开拓要求
| 市场 | 开拓费用 | 持续时间 | 维护要求 |
| ---- | ---- | ---- | ---- |
| 区域 | 1M | 1年 | 每年最少投入1M维持，否则视为放弃 |
| 国内 | 2M | 2年 | 每年最少投入1M维持，否则视为放弃 |
| 亚洲 | 3M | 3年 | 每年最少投入1M维持，否则视为放弃 |
| 国际 | 4M | 4年 | 每年最少投入1M维持，否则视为放弃 |

#### 6.1.3 核心规则
- 新市场开拓需按对应费用和时间投入，全部投入完成后方可接单；
- 已进入的市场需每年投入1M维持，未达标则自动放弃该市场运营资格。

### 6.2 销售会议与订单争取
#### 6.2.1 选单顺序规则
1. 由上年在该市场的订单价值决定市场领导者，市场领导者最先选择订单；
2. 非市场领导者按产品的广告投入量多少，依次选择订单；
3. 若同一产品有多家企业广告投入相同，则按该市场上全部产品的广告投入量决定选单顺序；
4. 若市场广告投入量仍相同，则按上年订单销售额的排名决定顺序；
5. 以上条件均相同时，通过招标方式选择订单。

#### 6.2.2 销售会议流程
每年初各企业销售经理与客户见面召开销售会议，结合市场地位、产品广告投入、市场广告投入、市场需求及竞争态势，按上述规则依次选择订单。

### 6.5 生产线购买、转产与维护、出售
#### 6.5.1 生产线核心参数
| 生产线类型 | 购买价格 | 安装周期 | 生产周期 | 转产周期 | 转产费用 | 维护费用 | 出售残值 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 手工线 | 5M | 无 | 3Q | 无 | 无 | 1M/年 | 1M |
| 半自动 | 8M | 2Q | 2Q | 1Q | 1M | 1M/年 | 2M |
| 全自动 | 16M | 4Q | 1Q | 2Q | 4M | 1M/年 | 4M |
| 柔性线 | 24M | 4Q | 1Q | 无 | 无 | 1M/年 | 6M |

#### 6.5.2 生产线通用规则
1. 所有生产线可生产全部产品，加工费统一为1M/产品；
2. 购买规则：投资新生产线时按安装周期平均支付投资，全部投资到位的下一个季度领取产品标识，开始生产；
3. 转产规则：现有生产线转产新产品需按对应转产周期执行并支付转产费用，最后一笔支付到期一个季度后方可更换产品标识；
4. 维护规则：当年在建的生产线和当年出售的生产线无需支付维护费；
5. 出售规则：出售时，若生产线净值小于残值，将净值转换为现金；若净值大于残值，将残值部分转换为现金，差额部分计入综合费用（其他）；
6. 折旧规则：每年按生产线净值的1/3取整计算折旧；当年建成的生产线不提折旧；当生产线净值小于3M时，每年提1M折旧。

### 6.6 产品生产与原材料采购
#### 6.6.1 原材料采购规则
1. 采购流程：根据上季度所下采购订单接受相应原料入库，并按规定付款或计入应付款；
2. 订货标识：用空桶表示原材料订货，放置于对应订单上；
3. 采购提前期：R1、R2订购需提前1个季度；R3、R4订购需提前2个季度。

#### 6.6.2 产品生产规则
1. 生产启动：开始生产时需按产品结构要求将原料放在生产线上，并支付加工费（1M/产品）；
2. 生产限制：各生产线不能同时生产两个产品；
3. 原料要求：上线生产必须具备对应原料，否则需“停工待料”。

#### 6.8.2 市场开发要求
| 市场 | 完成时间 | 投资规则 |
| ---- | ---- | ---- |
| 区域 | ≥1年 | 1M/年，每年最多投资1M，不允许加速，可中断 |
| 国内 | ≥2年 | 同上 |

| 国际 | ≥4年 | 同上 |

#### 6.8.3 通用规则
1. 投资执行：市场开发投资按年度支付，可同时开发多个市场；ISO两项认证投资可同时进行或延期；
2. 资格获取：市场开发完成后持开发费用凭证领取市场准入证，ISO认证完成后领取对应资格证；
3. 费用核算：研发投资与认证投资均计入当年综合费用。

### 6.9 融资贷款与资金贴现
#### 6.9.1 贷款与贴现参数
| 类型 | 贷款/贴现时间 | 额度/条件 | 年息/贴现率 | 还款方式 |
| ---- | ---- | ---- | ---- | ---- |
| 长期贷款 | 每年年末 | 权益的2倍，贷款额为20的倍数 | 10% | 年底付息，到期还本，最长期限5年 |
| 短期贷款 | 每季度初 | 权益的2倍，贷款额为20的倍数 | 5% | 到期一次还本付息，期限1年 |
| 高利贷 | 任何时间 | - | 20% | 到期一次还本付息，期限1年（不足1年按1年计息） |
| 资金贴现 | 任何时间 | 视应收款额，金额为7的倍数 | 1:6（7M应收款贴现需支付1M贴现费） | 变现时贴息 |

#### 6.9.2 核心规则
- 计息规则：长期贷款每年需还利息；短期贷款及高利贷到期时还本付息；不足1年按1年计息；
- 贴现规则：有应收款时可随时进行资金贴现，不受应收款期限限制。

### 6.10 综合费用与折旧、税金
#### 6.10.1 综合费用构成
行政管理费、市场开拓费、产品研发费、ISO认证费、广告费、生产线转产费、设备维修费、厂房租金等，均计入综合费用。

#### 6.10.2 折旧计算
设备折旧按余额递减法计算：每年按生产线净值的1/3取整计算折旧；当年建成的生产线不提折旧；当生产线净值小于3M时，每年提1M折旧。

#### 6.10.3 税金缴纳
每年所得税计入应付税金，在下一年初交纳。

## 七、实际模拟训练流程
### 7.1 前期准备
1. 分组分工：根据人员定位完成小组组建，明确各成员岗位职责；
2. 规则学习：深入学习企业运营各项规则，确保全员掌握核心要求；
3. 初始状态熟悉：梳理模拟企业的生产、物流、财务、营销等初始数据，明确企业初始资源与能力。

### 7.2 模拟运营步骤
1. 战略制定：总经理牵头，结合市场情况与企业目标，制定年度发展战略（产品研发、市场开拓、生产扩张等）；
2. 年度规划：各部门基于总体战略，制定年度工作计划（营销部门制定广告投放计划、生产部门制定生产计划、采购部门制定采购计划、财务部门制定融资与预算计划）；
3. 季度运营：按季度执行运营流程，包括原料采购、生产组织、产品销售、财务核算等，记录运营数据；
4. 年度总结：年末进行财务结算、绩效分析，对比实际运营结果与计划目标，调整下一年度战略与计划；
5. 多年度循环：重复上述步骤，完成多个年度的模拟运营，积累企业管理经验。

### 7.3 过程记录与报告
1. 运营记录：实时记录企业运营过程中的关键数据（生产产量、销售订单、资金流向、库存变化等）；
2. 报告编制：模拟运营结束后，小组完成调研报告（分析市场情况、竞争态势、运营问题等），各成员完成个人学习报告（总结岗位职责履行情况、收获与体会）。
