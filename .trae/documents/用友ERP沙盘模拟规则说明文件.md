# 用友ERP沙盘模拟详细说明文件
## 三、沙盘模拟核心框架
### （一）模拟目标
通过还原企业全流程运营，培养学员的战略规划、团队协作、市场分析、财务管控等综合能力，助力学员向“懂技术又懂管理的复合型人才”发展。

### （二）整体流程
1. 组织准备工作；
2. 熟悉模拟企业概况；
3. 明确初始状态设定；
4. 掌握企业运营规则；
5. 开展实际模拟训练。

## 四、组织准备工作
### （一）人员定位与职责
#### 1. 总经理
- 制定企业中长期发展战略；
- 分析行业竞争格局与市场趋势；
- 确定年度经营核心指标；
- 制定产品、市场、融资等业务策略；
- 统筹全面预算管理；
- 协调管理团队成员协同工作；
- 开展企业绩效分析与业绩考评；
- 负责管理授权与运营总结。

#### 2. 财务主管
- 负责日常财务记账、登账及财务报表编制；
- 实施日常现金流量管理；
- 制定企业融资策略与资金调度方案；
- 开展成本费用控制与风险管理；
- 建立健全财务制度；
- 提供财务分析数据，协助总经理决策。

#### 3. 营销主管
- 开展市场调查与需求分析；
- 制定市场进入、品种发展、广告宣传策略；
- 编制年度销售计划；
- 参与客户谈判，争取订单资源（按简化规则执行，无竞单排序机制）；
- 签订销售合同并实施过程控制；
- 负责按时发货与应收款管理；
- 开展销售绩效分析与优化。

#### 4. 生产主管
- 统筹产品研发管理与管理体系认证；
- 规划固定资产投资与生产线布局（厂房配置固定，无需购买/租赁/出售）；
- 编制年度及季度生产计划；
- 平衡生产能力与生产负荷；
- 负责生产车间管理与产品质量保证；
- 管控成品库存与产品外协业务。

#### 5. 供应主管
- 依据生产计划编制采购计划；
- 与供应商谈判并签订采购合同；
- 监控采购执行过程与到货验收；
- 开展仓储管理与采购支付决策；
- 协调与财务部的资金结算、与生产部的原料供应协同。

### （二）职能定位与组织架构
| 中心名称 | 核心职能 | 对应岗位 |
| ---- | ---- | ---- |
| 财务中心 | 财务核算、资金管理、融资规划、财务分析 | 财务主管、财务助理 |
| 生产中心 | 生产计划、产能管理、产品研发、质量管控 | 生产主管 |
| 物流中心 | 原料采购、仓储管理、库存管控、到货验收 | 供应主管 |
| 营销与规划中心 | 市场开拓、广告投放、订单争取、战略规划 | 营销主管 |
| 决策层 | 战略制定、团队协调、全局管控 | 总裁CEO（总经理） |

### （三）团队协作要求
- 心态开放：积极接纳不同意见，拥抱创新思路；
- 亲历亲为：主动参与各项运营环节，不推诿责任；
- 团队协作：各岗位分工明确、高效配合，共享信息；
- 换位思考：理解其他岗位工作难点，相互支持。

## 七、企业运营规则（核心，修正后）
### （一）市场划分与市场准入
#### 1. 市场层级
本地市场（已准入）、区域市场、国内市场（删除亚洲、国际市场）。

#### 2. 开拓规则
| 市场类型 | 开拓费用 | 持续时间 | 维护要求 |
| ---- | ---- | ---- | ---- |
| 区域市场 | 1M/年 | ≥1年 | 无需投入维持费用，开拓完成后永久保留准入资格 |
| 国内市场 | 2M（分2年投入，1M/年） | ≥2年 | 无需投入维持费用，开拓完成后永久保留准入资格 |

#### 3. 关键说明
- 市场开拓投资按年度支付，可同时开发多个市场，但单个市场每年最多投资1M，不允许加速投资，允许中断；
- 需完成全部开拓投资后，领取市场准入证方可参与该市场业务开展（无竞单机制，按市场需求直接对接订单）；
- 开拓完成后无需持续投入维持费用，市场资格永久有效。

### （二）销售与订单管理（简化后）
- 广告投入：按年度规划广告预算，投入金额仅作为市场推广参考，不影响订单获取优先级；
- 订单获取：基于已开拓市场的需求容量，结合企业生产能力与产品竞争力，直接对接客户获取订单；
- 订单执行：按合同约定时间发货，及时跟进应收款回收，确保资金流稳定。

### （三）生产线购买、转产与维护、出售（修正后）
#### 1. 生产线核心参数（保留原有类型，新增固定初始配置）
| 生产线类型 | 购买价格 | 安装周期 | 生产周期 | 转产周期 | 转产费用 | 年维护费用 | 出售残值 | 初始配置 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 手工线 | 5M | 无 | 3Q | 无 | 无 | 1M/年 | 1M | 无 |
| 半自动线 | 8M | 2Q | 2Q | 1Q | 1M | 1M/年 | 2M | 无 |
| 全自动线 | 16M | 4Q | 1Q | 2Q | 4M | 1M/年 | 4M | 2条（大、小厂房各1条） |
| 柔性线 | 24M | 4Q | 1Q | 无 | 无 | 1M/年 | 6M | 无 |

#### 2. 操作规则（修正后）
- 适用范围：所有生产线可生产全部产品，加工费统一为1M/产品；
- 购买：如需新增生产线，按安装周期平均支付投资，全部投资到位的下一季度领取产品标识，启动生产；
- 转产：需完成转产周期并支付转产费用，最后一笔支付到期后下一季度可更换产品标识；
- 维护：当年在建或当年出售的生产线无需支付维护费；初始配置的2条全自动生产线按年支付维护费；
- 出售：如需出售生产线，① 生产线净值＜残值：净值全额转换为现金；② 生产线净值＞残值：残值部分转现金，差额计入综合费用（其他）；
- 删除折旧：生产线无需计提折旧，固定资产价值保持初始购买价或净值不变。

### （四）产品生产与原材料采购
#### 1. 原材料采购规则
| 原料类型 | 订购提前期 | 入库与付款方式 |
| ---- | ---- | ---- |
| R1 | 1个季度 | 上季度下订单，本季度入库，按规定付款或计入应付款 |
| R2 | 1个季度 | 上季度下订单，本季度入库，按规定付款或计入应付款 |
| R3 | 2个季度 | 前两个季度下订单，本季度入库，按规定付款或计入应付款 |
| R4 | 2个季度 | 前两个季度下订单，本季度入库，按规定付款或计入应付款 |

#### 2. 采购操作
- 用空桶表示原材料订货，放置在对应订单位置；
- 入库时需验收，确保原料数量与订单一致；
- 初始无原料库存，需根据生产计划提前下达采购订单。

#### 3. 产品生产规则
- 生产启动条件：生产线已领取对应产品标识、原料齐备、支付加工费（1M/产品）；
- 生产限制：单条生产线不能同时生产两种及以上产品；
- 停工规则：原料短缺时必须“停工待料”，待原料补齐后恢复生产；
- 生产示例（P2产品）：需投入R1+R2原料，加工费1M，全自动生产线生产周期1Q。

### （六）市场开发和ISO认证
#### 1. ISO认证规则
| 认证类型 | 建立时间 | 年投资金额 | 投资处理 |
| ---- | ---- | ---- | ---- |
| ISO9000 | ≥2年 | 1M/年 | 计入当年综合费用，完成后领取资格证 |
| ISO14000 | ≥3年 | 1M/年 | 计入当年综合费用，完成后领取资格证 |

#### 2. 补充说明
- 两项认证可同时进行或延期，投资可中断；
- 认证资格是部分高端市场或大额订单的合作前提。

### （七）融资贷款规则（删除高利贷、资金贴现）
#### 1. 贷款规则
| 贷款类型 | 贷款时间 | 贷款额度 | 年息 | 还款方式 | 期限 | 其他要求 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 长期贷款 | 每年第一和第四季度 | 0-40M | 10% | 年底付息，到期还本 | 3年 | 贷款金额为20的倍数，最高40M |

#### 2. 禁止行为
- 禁止民间借贷、层层转包等违法行为；
- 禁止未按规定期限还款，否则按违约处理（影响企业信用评级）。

### （八）综合费用（删除折旧、税金）
#### 1. 综合费用构成
行政管理费（年底固定缴纳1M）、市场开拓费、产品研发费、ISO认证费、广告费、生产线转产费、设备维修费（厂房为固定配置，无需支付租金）等。

### （九）本次运营特别说明（最终修正版）
1. 产品结构：P1=R1+1M（加工费），P2=R1+R2+1M（加工费）；
2. 市场设定：仅保留本地、区域、国内市场，区域市场（第二年可准入）、国内市场（第三年可准入），开拓完成后无需维持费用；
3. 广告规则：广告投入仅作为市场推广支持，不影响订单获取优先级；
4. 厂房与生产线配置：大厂房、小厂房各1个（固定不可交易），每个厂房配套1条全自动生产线；
5. P2研发补充：周期6Q，总投资6M，可按季度分期支付；
6. 行政费用：年底固定缴纳1M，计入综合费用；
7. 初始状态：现金40M，长期负债0M，在制品、成品、原料及原料库存均为0，应缴税0M；
8. 删除机制：市场竞单机制、厂房购买/租赁/出售机制、资金贴现规则、税金规则、折旧规则、高利贷、亚洲市场、国际市场；
9. 融资限制：仅支持长期贷款、短期贷款，贷款金额为20的倍数，长期贷款最高40M（3年期），短期贷款最高40M（1年期）。
